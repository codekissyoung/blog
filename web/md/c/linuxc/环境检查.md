# linux 系统调用处理调用失败
- 系统调用失败一般返回`-1`,并且将全局变量 `errno` 设置为相应的错误码
```c
cnt = read(fd,buf,numbytes);
if(cnt == -1){
    if(errno == EINTR)
        printf(stderr,"read was interrupted by ad signal\n");
    else
        printf("unknown error\n");
}

fd = open(pathname,flags,mode);
if(fd == -1){
    perror("open");
    exit(EXIT_FAILURE);
}
```
