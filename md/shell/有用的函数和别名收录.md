执行一个命令 直到命令运行成功
================================================================================
```bash
repeat (){
    while : ; do  # : 总会返回0的退出码 比 true 快
        $@ && return ;
        sleep 5; # 如果失败了,那么就再延迟5秒，再循环执行命令 
    done
}

#　eg.
repeat wget -c http://www.xunlei.com/software-aa.tar.gz
```
